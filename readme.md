User authentication and authorization app:
- SignUp
- Login
- Logout
- Forget password
- Update password
- Get user profile
- Update user profile

Technology stack
- django 3.0
- python 3.6

Database
- postgresql

Setting up app 
- clone the app

running the app
- need postgresql installed in your system
- go to directory where there is manage.py file
- run using command (python manage.py runserver)
